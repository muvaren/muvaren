## Featured Project

<div align="center">

### AI-Powered USB Behavioral Threat Intelligence Platform
*Academic Final Year Project*

</div>

**Purpose**
A SOC-style security monitoring platform built to detect, log, and investigate suspicious USB device activity.

**Architecture**
A Python and Flask backend serving a SOC-style dashboard, backed by SQLite for structured event and log storage, developed and version-controlled on Ubuntu Linux with Git.

**Security Features**
- Real-time monitoring of USB device activity
- Behavioral analysis to flag suspicious device behavior
- Forensic log generation for security investigations
- SOC-style dashboard for tracking events and security alerts

**Tech Stack**

`Python` `Flask` `SQLite` `Git` `Ubuntu Linux`
